SaaS Chat Widget NPM Example

Using the NPM Package

This example demonstrates how to use the SaaS Chat Widget from npm.

Installation

npm install saas-chat-widget

Usage

import SaaSChatWidget from 'saas-chat-widget';

const chatWidget = new SaaSChatWidget({
  apiKey: 'YOUR_API_KEY',
  organization: 'your-organization',
  apiUrl: 'http://localhost:5030',
  socketUrl: 'http://localhost:5030',
  debug: true
});
